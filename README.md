# NeonLauncher

Cyberpunk AAA PC Game Launcher built with C++ and Qt6.

## Features

- Steam-like UI
- Cyberpunk animations
- Game scanner
- Game launcher
- SQLite library
- Mod manager
- Marketplace system
- RGB sync
- Overlay system
- AI recommendation engine

## Tech Stack

- C++20
- Qt 6
- QML
- SQLite
- CMake
- WinAPI

## Build

```bash
cmake -B build
cmake --build build --config Release
